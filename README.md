# Fscan输出文件整理
对Fscan扫描结果进行整理
动态显示和生成表格，不会显示和生成没有结果的项

## 使用
pip install -r requirements.txt `安装个别库失败的话 试试删除requirements中的版本限制`

python FscanOutputBeautif.py result.txt `推荐Py3.9+运行`
